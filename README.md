# Maximizing Marketing Campaigns through Data-Driven Customer Segmentation

## 📌 Executive Summary

Công ty bán lẻ đang gặp vấn đề chi phí Marketing cao, tỷ lệ chốt đơn thấp. Mục tiêu của dự án là sử dụng dữ liệu hành vi và nhân khẩu học để định vị lại tệp khách hàng lõi.

## 🛠 Tech Stack 

Báo cáo: RMarkdown -> HTML.

Ngôn ngữ: R.

Thư viện:

- Xử lý dữ liệu: `tidyverse, dplyr, janitor, lubridate`.

- Trực quan hoá: `ggplot2, ggpubr, gridExtra, scales.`

- Báo cáo: `knitr, skimr.`

Kỹ thuật: Phân tích Khám phá (EDA), Xử lý Outliers, Phân cụm khách hàng dựa theo phân vị.
